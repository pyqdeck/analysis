Topic Importance & Frequency

Tier 1: The Guaranteed Marks Zone

These topics are on almost every paper. Master them tonight for maximum return on investment.

Must-Know Graph Theory Proofs

  • 1.A tree with 'n' vertices has '(n-1)' edges. (Proof by induction).
  • 2.Sum of degrees is 2 * |E| (Handshaking Lemma).
  • 3.Graph remains connected after removing edge 'e' IFF 'e' is in a circuit.
  • 4.Every tree with two or more vertices is 2-chromatic.

Must-Know Algebraic Proofs

  • 1.The one-step subgroup test (a * b⁻¹ ∈ H).
  • 2.Every group of prime order (e.g., 3, 5, 7) is cyclic. (Use Lagrange's).
  • 3.State and Prove Lagrange's Theorem.
  • 4.Prove De Morgan's Laws for sets.

Must-Solve Problem Types

  • 1.Inclusion-Exclusion Principle (e.g., swim/jog, faulty cars).
  • 2.Proof by Induction (e.g., `1.1! + ... + n.n! = (n+1)!-1`).
  • 3.Pigeonhole Principle (e.g., `at least ⌈N/K⌉` share...).
  • 4.Proof by Contradiction (e.g., √2 is irrational).

Tier 2: The High-Probability Zone

If you're solid on Tier 1, these are your next targets. They appear very frequently.

Logic & Boolean Algebra

  • Prove a statement is a tautology (using truth tables or logical equivalences).
  • Find PDNF (Sum of Products) and PCNF (Product of Sums) of a function.

Graph Algorithms in Action

  • Apply Dijkstra's algorithm to find the shortest path in a weighted graph.
  • Apply BFS for shortest path in an unweighted graph and for traversal.
  • Determine Chromatic Number of a given graph.

Structure Verification Problems

  • Prove a given relation is an Equivalence Relation.
  • Prove a set with custom operations (like a*b = a+b-ab) forms a Group, Ring, or Semigroup.
  • Find roots of a polynomial over a Ring (e.g., Z₁₀).

Final Exam Day Strategy
